# Quiz Management System

MERN stack app for creating and taking quizzes with auto-evaluation.

## Setup

1. **Start MongoDB:**
```bash
mongod --dbpath C:\data\db
```

2. **Start Backend:**
```bash
cd backend
npm install
npm start
```

3. **Start Frontend:**
```bash
cd frontend
npm install
npm run dev
```

4. **Open:** `http://localhost:3000`

## Login

- **Trainer:** `trainer@test.com`
- **Learner:** `learner@test.com`

Any email works - no password needed!

## Features

- Create quizzes (MCQ, True/False)
- Auto-grading with instant results
- Timed quizzes with pass/fail criteria
- Attempt history and stats
